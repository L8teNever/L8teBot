# L8teBot

L8teBot is a powerful, modular Discord bot with a web dashboard, Twitch integration, leveling system, and more.

## Quick Start

### Docker (Recommended)
1. **Pull or Build**: Download the `docker-compose.yml` or clone the repo.
2. **Run**: `docker-compose up -d`
3. **Setup**: Open `http://localhost:5000` in your browser to configure your bot tokens on the first run.

### Local Installation
1. Install Python 3.11+.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
4. Follow the setup wizard in your browser at `http://localhost:5000`.
